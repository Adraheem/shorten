# URL Shortener
This is a Java Spring boot server application to shorten long URLs into short and customizable links.

Technologies used:
* `Springboot`
* `Spring Security`
* `Swagger docs`
* `JWToken`
* `PostgreSQL`

## Important links
To clone project 
```
git clone https://github.com/Adraheem/shorten.git
```
To view documentation:\
